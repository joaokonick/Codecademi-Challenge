Tables tags

Tag Name Description
<table> Table The wrapper element for all Html tables
<thead> Table Head The set of rows defining the headers in a table
<tbody> Table Body The set of rows containing actual table data
<tr> Table Row The table row container
<td> Table Data The table cell containing actual data
<tfoot> Table Foot The set of rows defining the footer in a table

Table attributes

Attributes Name Description
colspan Column Span Defines how many column a td element should span
rowspan Row Span Defines how many rows a <td> elements shouls span
